# libre-sealdice-ui

> 海豹核心的前端实现

## 说明

本仓库为海豹核心（AGPLv3 分叉）的前端实现，使用 Vue3 + ElementPlus 开发。

后端仓库：[libre-sealdice-core](https://github.com/sealdice/libre-sealdice-core)

分叉相关见上述仓库，该仓库包含对应的前端入口（如创建 LagrangeGo 相关连接），故同样分叉保留。

但该仓库本身不受 AGPLv3 传染，依然为 MIT 协议开源。

## 部署

```bash
npm install
```

### 开发环境

```bash
npm run dev
```

### 生产环境

```bash
npm run build
```
